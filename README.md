## 구성

* Core
  * 각 모듈별 도메인
* Mail
  * AWS SES를 이용한 메일 기능 
* Batch
  * 마이그레이션 등 배치 작업
* Static Uploader
  * AWS S3 정적파일 업로더
  * 이미지, 아카이브용 Html 등  
* Web
  * 메일 구독 등록
  * 페이스북 Web Hook
* Admin
  * 메일 발송용 컨텐츠 관리
