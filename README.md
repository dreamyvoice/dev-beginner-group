## 구성

* Core
* Mail
  * AWS SES를 이용한 메일 기능 
* Batch
  * 정기 메일 발송
  * 기타 배치 작업
* Static Uploader
  * AWS S3 정적파일 업로더  
* Web
  * 메일 구독 등록
  * 페이스북 Web Hook
* Admin
  * 메일 발송용 컨텐츠 관리
